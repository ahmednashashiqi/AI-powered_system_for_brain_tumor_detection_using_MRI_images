Help & Tips

Quick guidance on using the app
  • Go to Analyze, upload a single MRI slice (.png/.jpg/.tif).
  • Optionally type a question to steer the description (e.g., “Summarize likely findings”).
  • Enable Advanced Report to generate Grad-CAM heatmap (requires classifier checkpoint).
  • Enable Use RAG to enrich the explanation with retrieved references (requires built index).
  • See past runs in History. You can refresh or clear the list.

Backend endpoints: POST /analyze, GET /api/history, POST /api/history/clear, GET /healthz.
